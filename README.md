# Universal Web Scraper with Brand Psychology Analysis

A powerful web scraping tool that extracts content from any website, handles authentication automatically, and analyzes brand psychology using Google's Gemini AI.

## Features

- Scrapes any website (Instagram, Facebook, etc.)
- Automatic authentication handling with cookie storage
- Multiple screenshots with scrolling
- Extracts headlines, images, metadata, and links
- Brand psychology analysis using Gemini AI
- Organized folder structure per URL
- Process multiple URLs in one run
